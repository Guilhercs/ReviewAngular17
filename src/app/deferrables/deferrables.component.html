Deferrables

Idle Deferrable (Default)

Example:

@defer (on idle) { } @placeholder {

preview view

} @loading {

Loading...

} @error {

loading failed

}

Interaction Defer

@defer (on interaction(onClick)) { } @placeholder {

preview view

} @loading {

Loading...

} @error {

loading failed

}

Deferrable When

@defer (when isTrue ) {

page loaded

}
